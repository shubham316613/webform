install node,express,mongoose,nodemon using following cmds 
npm init (installs node)
node install express mongoose nodemon
express gives us frontend js, mongoose installs the all mongodb connectivity requirements, nodemon helps to refresh the running node app automatically once changes are made so that you don't need to refresh node app manually using node index.js cmd.
